Everything Local. Nothing Leaked.

Your code stays on your machine. Your agents stay in bounds.

🦙

Local LLMs via Ollama

Run qwen3, mistral, or any model locally. ~1.3 GB RAM for 1.7B params. No API keys, no cloud, no data exfiltration.

🛡️

AgentGuard Governance

Policy-as-code in agentguard.yaml. Block force-push, restrict file writes, enforce timeouts. Monitor mode → enforce mode when ready.

⚡

Script-Based. No Daemons.

Each agent is a bounded script that runs and exits. Cron-safe. No background processes, no distributed systems, no complexity.

🧠

Memory-Optimized

Pluggable memory layer ready for Google A2A, MemGPT, or custom context compaction. Swap implementations with zero refactor.

🔌

Framework Ready

Adapter stubs for DeepAgents and OpenCode. When they ship, plug them in — the interfaces are already defined.

📊

Observable

Every run writes structured output. Token tracking, execution timing, governance decisions — all logged to disk and optionally to AgentGuard Cloud.

Built-in Agents

Simple, deterministic, bounded. Each one does one thing well.

Agent Input Output Type

The Full Stack

Eight technologies. One governed agent runtime. Here's how they fit together.

INPUT 📝 Your Prompt "Analyze this repo for test gaps"
↓
COMPRESS ⚡ RTK (Rust Token Killer) Strips 70-90% of terminal noise before the LLM sees it
↓
MEMORY 🧠 TurboQuant (Google, ICLR 2026) 6x KV cache compression — run 14B models on 8GB Macs
↓
MODEL 🦙 Ollama Local inference — qwen3, mistral, any GGUF model
↓
AGENT 🔥 ShellForge Agent execution loop — input → prompt → model → output → save
↓
GOVERN 🛡️ AgentGuard Policy-as-code — allow/deny every tool call, full audit trail
↓
SCAN 🐾 DefenseClaw (Cisco, RSA 2026) Supply chain security — scan skills, verify MCP servers, AI BoM
↓
SANDBOX 🔒 OpenShell (NVIDIA, GTC 2026) Kernel-level isolation — Landlock + Seccomp, signed audit logs
