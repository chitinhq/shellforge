Everything Local. Nothing Leaked.

Your code stays on your machine. Your agents stay in bounds.

🦙

Local LLMs via Ollama

Run qwen3, mistral, or any model locally with Metal GPU acceleration. ~1.3 GB RAM for 1.7B params. No API keys, no cloud, no data exfiltration.

🛡️

AgentGuard Governance

Policy-as-code in agentguard.yaml. Block force-push, restrict file writes, enforce timeouts. Monitor mode → enforce mode when ready.

⚡

Single Binary. No Daemons.

One Go binary, zero dependencies. Each agent command runs and exits. Cron-safe. No background processes, no distributed systems, no complexity.

🧠

Memory-Optimized

RTK strips 70-90% of terminal noise before the LLM sees it. KV cache quantization doubles agent capacity on 8GB Macs.

🔌

Multi-Driver Governance

Govern any CLI agent — Crush (local), Claude Code, Copilot CLI, Codex, Gemini. One governance policy across all drivers.

🔄

Dagu Orchestration

shellforge swarm — YAML DAG workflows with web dashboard. Schedule agents, define dependencies, run a 24/7 swarm on your Mac or Linux box.
